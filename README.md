# aishwarya-lokhande-portfolio

Personal portfolio for **Aishwarya Lokhande** — Software Engineer building fintech backend systems with Java & Spring Boot in Pune, India.

## Stack
- **Next.js 14** (App Router) · **TypeScript** · **Tailwind CSS**
- Fonts: Playfair Display (display) · Epilogue (body) · JetBrains Mono (code)
- Zero env vars required

## Sections
| Section    | Content |
|------------|---------|
| Hero       | Animated role switcher, stat strip, CTAs |
| About      | Engineering philosophy, 4 specialty pillars |
| Experience | Finarkein Analytics + Solve All Solutions (timeline) |
| Projects   | Pariksha (featured) + 3 professional work cards |
| Skills     | 6 groups from resume + education + achievements + coding profiles |
| Contact    | Email / LinkedIn / phone + "What I bring" pitch card |

## Local Setup

```bash
git clone https://github.com/yourusername/aishwarya-lokhande-portfolio.git
cd aishwarya-lokhande-portfolio
npm install
npm run dev
# → http://localhost:3000
```

No environment variables needed.

## Build

```bash
npm run build && npm start
```
